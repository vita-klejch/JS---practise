This simple, easy to use, responsive app will help you to plan your events
In this free to use app you can:
  • Create new events
  • Sort your events
  • Modify you events

Please press the button Create new event, complete the form, and press the button Save
The current time and date are displayed for your convenience

Please press the button Sort by: and sort your events by:
  • importance
  • date of creation
  • date of event

Please press the button Show more details on you event
  • press the button complete or return to change the display of the event
  • press the button Edit event and the filled out form will open
  • press the button Delete event to completely remove you event
